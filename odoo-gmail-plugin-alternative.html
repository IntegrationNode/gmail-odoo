Does the Odoo Gmail Plugin work on self-hosted / on-premise Odoo?

Short answer: only if you edit its source code and deploy it yourself. This is the real difference.

To run the OFFICIAL plugin on self-hosted Odoo (custom domain), Odoo's own docs require you to:
  1. Download the mail-client-extensions repo from GitHub
  2. Edit login.ts to remove the odoo.com domain check
  3. Edit appsscript.json to whitelist your server domain
  4. Deploy it as your own Google Apps Script project
  5. Publish from manifest and install the add-on
With our extension:

Enter your Odoo URL. Done. No code, no deployment. Works on Community and Enterprise, any domain, Odoo 14–19.

These are the steps from Odoo's own documentation (mail_plugins/gmail) — we're describing their process, not inventing it.
