¿Funciona el plugin de Gmail de Odoo en Odoo self-hosted / on-premise?

Respuesta corta: solo si editas su código fuente y lo despliegas tú mismo. Esta es la verdadera diferencia.

Para usar el plugin OFICIAL en Odoo self-hosted (dominio propio), la propia documentación de Odoo exige:
  1. Descargar el repositorio mail-client-extensions desde GitHub
  2. Editar login.ts para quitar la comprobación del dominio odoo.com
  3. Editar appsscript.json para autorizar el dominio de tu servidor
  4. Desplegarlo como tu propio proyecto de Google Apps Script
  5. Publicarlo desde el manifest e instalar el complemento
Con nuestra extensión:

Introduce la URL de tu Odoo. Listo. Sin código, sin despliegue. Funciona en Community y Enterprise, cualquier dominio, Odoo 14–19.

Estos son los pasos de la propia documentación de Odoo (mail_plugins/gmail) — describimos su proceso, no lo inventamos.
